# Decode Slanted Transposition Cipher

## 🧠 Problem Summary

A string is encoded using a slanted transposition cipher with a fixed
number of rows.

-   The text is filled diagonally (top-left → bottom-right) in a matrix.
-   Then it is read row-wise to form the encoded text.

Your task is to reconstruct the original string.

------------------------------------------------------------------------

## 📌 Approach

1.  Compute number of columns:

        cols = len(encodedText) // rows

2.  Rebuild the matrix row-wise.

3.  Traverse diagonally:

    -   Start from each column in the first row
    -   Move `(i+1, j+1)` until out of bounds

4.  Remove trailing spaces.

------------------------------------------------------------------------

## ✅ Python Solution

``` python
class Solution:
    def decodeCiphertext(self, encodedText: str, rows: int) -> str:
        if rows == 1:
            return encodedText

        n = len(encodedText)
        cols = n // rows

        # Step 1: Build matrix
        matrix = []
        index = 0
        for i in range(rows):
            matrix.append(list(encodedText[index:index+cols]))
            index += cols

        # Step 2: Traverse diagonally
        result = []

        for start_col in range(cols):
            i, j = 0, start_col
            while i < rows and j < cols:
                result.append(matrix[i][j])
                i += 1
                j += 1

        # Step 3: Remove trailing spaces
        return ''.join(result).rstrip()
```

------------------------------------------------------------------------

## 🔍 Example

**Input:**

    encodedText = "ch   ie   pr"
    rows = 3

**Output:**

    "cipher"

------------------------------------------------------------------------

## ⏱️ Complexity

-   Time: O(n)
-   Space: O(n)

------------------------------------------------------------------------

## 💡 Key Insight

Encoding = diagonal write + row read\
Decoding = row rebuild + diagonal read
