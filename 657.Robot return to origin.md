# Robot Return to Origin 🚀

## Problem Statement

There is a robot starting at position (0, 0) on a 2D plane. Given a
sequence of moves, determine if the robot returns to the origin.

### Moves:

-   'U' → Up
-   'D' → Down
-   'L' → Left
-   'R' → Right

Return `True` if the robot returns to (0, 0), otherwise `False`.

------------------------------------------------------------------------

## Approach 🧠

-   Initialize position `(x, y) = (0, 0)`
-   Traverse each move:
    -   'U' → y += 1
    -   'D' → y -= 1
    -   'R' → x += 1
    -   'L' → x -= 1
-   Check if `(x, y) == (0, 0)`

------------------------------------------------------------------------

## Python Code 🧾

``` python
class Solution:
    def judgeCircle(self, moves: str) -> bool:
        x, y = 0, 0

        for move in moves:
            if move == 'U':
                y += 1
            elif move == 'D':
                y -= 1
            elif move == 'R':
                x += 1
            elif move == 'L':
                x -= 1

        return x == 0 and y == 0
```

------------------------------------------------------------------------

## Optimized Approach ⚡

``` python
class Solution:
    def judgeCircle(self, moves: str) -> bool:
        return moves.count('U') == moves.count('D') and                moves.count('L') == moves.count('R')
```

------------------------------------------------------------------------

## Examples 🧪

``` python
sol = Solution()
print(sol.judgeCircle("UD"))  # True
print(sol.judgeCircle("LL"))  # False
```

------------------------------------------------------------------------

## Conclusion ✅

-   If opposite moves cancel each other → robot returns to origin
-   Otherwise → it does not
