# Product Matrix of a Grid

## 🧩 Problem

Given a `0-indexed` 2D integer matrix `grid` of size `n × m`, construct a matrix `p` such that:

- Each element `p[i][j]` is the product of **all elements in grid except `grid[i][j]`**
- The result is taken **modulo 12345**

---

## 💡 Key Insight

A brute-force approach would compute the product for each cell separately:

- Time Complexity: **O((n × m)²)** ❌ (too slow)

👉 Instead, we optimize using:

> 🔁 **Prefix + Suffix Product Technique**

---

## ⚠️ Why Not Use Division?

- Modulo = `12345` (not prime)
- Division under modulo is **not reliable**

👉 So we avoid division and use prefix/suffix method ✅

---

## 🧠 Approach

### Step 1: Flatten the Grid

Convert 2D grid into 1D array:
[grid[0][0], grid[0][1], ..., grid[n-1][m-1]]

### Step 2: Prefix Product
prefix[i] = product of all elements before index i


---

### Step 3: Suffix Product


suffix[i] = product of all elements after index i


---

### Step 4: Compute Result


result[i] = (prefix[i] × suffix[i]) % 12345


Then reshape back to `n × m` matrix.

---

## ✅ Python Solution (LeetCode Ready)

```python
class Solution:
    def constructProductMatrix(self, grid):
        MOD = 12345
        
        n, m = len(grid), len(grid[0])
        
        # Step 1: Flatten grid
        arr = []
        for row in grid:
            arr.extend(row)
        
        k = len(arr)
        
        # Step 2: Prefix products
        prefix = [1] * k
        for i in range(1, k):
            prefix[i] = (prefix[i-1] * arr[i-1]) % MOD
        
        # Step 3: Suffix products
        suffix = [1] * k
        for i in range(k-2, -1, -1):
            suffix[i] = (suffix[i+1] * arr[i+1]) % MOD
        
        # Step 4: Build result matrix
        result = []
        idx = 0
        
        for i in range(n):
            row = []
            for j in range(m):
                val = (prefix[idx] * suffix[idx]) % MOD
                row.append(val)
                idx += 1
            result.append(row)
        
        return result
```

Input:
grid = [[1,2],

Output:

🧪 Example 2

        [2],

 [0],
 [0]]
