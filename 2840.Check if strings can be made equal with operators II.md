## 🧩 Problem Statement

You are given two strings `s1` and `s2` of length `n`.

### 🔁 Allowed Operation

You can perform the following operation any number of times:

-   Choose indices `i < j` such that:

        (j - i) is even

-   Swap `s[i]` and `s[j]`

------------------------------------------------------------------------

## 🎯 Goal

Return **true** if you can make `s1` equal to `s2`, otherwise return
**false**.

------------------------------------------------------------------------

## 🧪 Examples

### Example 1

    Input:
    s1 = "abcdba"
    s2 = "cabdab"

    Output:
    true

------------------------------------------------------------------------

### Example 2

    Input:
    s1 = "abe"
    s2 = "bea"

    Output:
    false

------------------------------------------------------------------------

## 💡 Key Insight

If `(j - i)` is even → both indices have the **same parity**.

👉 So swaps are only allowed between: - Even ↔ Even indices - Odd ↔ Odd
indices

------------------------------------------------------------------------

## 🔗 Important Observation

This means: - All **even indexed characters** can be rearranged among
themselves - All **odd indexed characters** can be rearranged among
themselves

But: ❌ You CANNOT move characters between even and odd positions

------------------------------------------------------------------------

## 🧠 Approach

1.  Split both strings into:

    -   Even index characters
    -   Odd index characters

2.  Sort both groups

3.  Compare:

    -   Even groups must match
    -   Odd groups must match

------------------------------------------------------------------------

## 🧪 Python Solution

``` python
class Solution:
    def checkStrings(self, s1: str, s2: str) -> bool:
        s1_even = sorted(s1[i] for i in range(0, len(s1), 2))
        s1_odd = sorted(s1[i] for i in range(1, len(s1), 2))
        
        s2_even = sorted(s2[i] for i in range(0, len(s2), 2))
        s2_odd = sorted(s2[i] for i in range(1, len(s2), 2))
        
        return s1_even == s2_even and s1_odd == s2_odd
```

------------------------------------------------------------------------

## ⏱ Complexity

  Operation             Complexity
  --------------------- ----------------
  Splitting + Sorting   O(n log n)
  Total                 **O(n log n)**

------------------------------------------------------------------------

## 🚀 Summary

-   Swaps only allowed within same parity indices
-   Treat as two independent groups:
    -   Even index group
    -   Odd index group
-   Compare sorted versions

------------------------------------------------------------------------

## 🎉 Final Thought

This problem is all about **parity grouping + invariants**.

If the groups match → ✅ possible\
If not → ❌ impossible
