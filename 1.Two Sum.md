# 🧠 Two Sum Problem - Explained in Detail

## ✅ Problem Statement

Given an array of integers `nums` and an integer `target`, return the **indices** of the two numbers such that they **add up to the target**.

### 🔒 Constraints
- Each input will have **exactly one solution**.
- You **cannot use the same element twice**.
- Return the answer in **any order**.

---

## 📥 Example Inputs and Outputs

| Input `nums`     | Target | Output | Explanation                      |
|------------------|--------|--------|----------------------------------|
| `[2,7,11,15]`     | `9`    | `[0,1]`| `2 + 7 = 9`                      |
| `[3,2,4]`         | `6`    | `[1,2]`| `2 + 4 = 6`                      |
| `[3,3]`           | `6`    | `[0,1]`| `3 + 3 = 6`                      |

---

## 🧑‍💻 Python Code

```python
class Solution:
    def twoSum(self, nums, target):
        num_map = {}  # value -> index

        for i, num in enumerate(nums):
            complement = target - num
            if complement in num_map:
                return [num_map[complement], i]
            num_map[num] = i
```

---

## 🔍 Step-by-Step Explanation

### 💡 Idea:
We use a **dictionary (`num_map`)** to store the number and its index as we iterate through the list.

### 🔁 Looping through the array:
```python
for i, num in enumerate(nums):
```
- `i`: Current index
- `num`: Current value

### 🧮 Compute the complement:
```python
complement = target - num
```
- For every number `num`, the **complement** is the number we need to find such that:
```
num + complement = target
```

### ❓ Check if the complement exists:
```python
if complement in num_map:
    return [num_map[complement], i]
```
- If we've seen the complement before (i.e., it's already in `num_map`), then we have found the solution.

### 🗂️ If not, store the number:
```python
num_map[num] = i
```
- We add the current number and its index to the dictionary, in case it's needed later as a complement.

---

## 🕒 Time & Space Complexity

| Complexity     | Description                                |
|----------------|--------------------------------------------|
| 🕒 Time        | O(n) – Single pass through the array       |
| 💾 Space       | O(n) – For storing elements in the hashmap |

---

## 🧪 Dry Run Example

### Example:
```python
nums = [2, 7, 11, 15], target = 9
```

| i | num | complement | num_map             | Action                    |
|---|-----|------------|---------------------|---------------------------|
| 0 | 2   | 7          | {}                  | Store `2: 0`              |
| 1 | 7   | 2          | {2: 0}              | Found! Return `[0, 1]`    |

---

## 🧠 Why This Works

- We only check each number once.
- At each step, we either:
  - Add the number to the map (so it can be used as a complement later), or
  - Use a previously seen number as the complement.

This is both **efficient** and **elegant**.

---

## 📌 Final Notes

- This approach only works if **exactly one solution exists**.
- The order of returned indices doesn't matter (e.g., `[0, 1]` or `[1, 0]` both are valid).
- Make sure not to use the **same element twice** (the map handles this naturally).

> ✅ This is the most efficient solution to the Two Sum problem using hashing in Python.
