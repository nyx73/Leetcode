# 🧮 3567. Minimum Absolute Difference in Sliding Submatrix

## 📌 Problem Statement

You are given an `m x n` integer matrix `grid` and an integer `k`.

For every contiguous `k x k` submatrix of `grid`, compute the **minimum absolute difference between any two distinct values** within that submatrix.

Return a 2D array `ans` of size `(m - k + 1) x (n - k + 1)`, where:

- `ans[i][j]` represents the minimum absolute difference in the submatrix with top-left corner `(i, j)`.

### ⚠️ Note:
- If all elements in a submatrix are the same, the answer is `0`.

---

## 🧠 Key Idea

To find the minimum absolute difference:

1. Extract all elements from the `k x k` submatrix
2. Remove duplicates (consider only distinct values)
3. Sort the values
4. Compute the minimum difference between adjacent elements

### 💡 Why sorting?
The smallest difference between any two numbers in a set will always be found between **adjacent elements in sorted order**.

---

## 🚀 Approach

For every valid starting position `(i, j)`:

1. Collect all values inside the `k x k` window
2. Convert to a `set` to remove duplicates
3. If only one unique value → answer = `0`
4. Otherwise:
   - Sort the values
   - Compute the minimum difference between consecutive elements

---

## ⏱ Time Complexity

- Number of submatrices: `(m - k + 1) * (n - k + 1)`
- For each submatrix:
  - Extraction: `O(k²)`
  - Sorting: `O(k² log k²)`

✅ Efficient enough for constraints (`m, n ≤ 30`)

---

## 🧑‍💻 Python Implementation

```python
from typing import List

class Solution:
    def minAbsDiff(self, grid: List[List[int]], k: int) -> List[List[int]]:
        m, n = len(grid), len(grid[0])
        ans = [[0] * (n - k + 1) for _ in range(m - k + 1)]

        for i in range(m - k + 1):
            for j in range(n - k + 1):
                values = set()
                
                # Collect elements in k x k submatrix
                for x in range(i, i + k):
                    for y in range(j, j + k):
                        values.add(grid[x][y])
                
                # If only one unique value
                if len(values) <= 1:
                    ans[i][j] = 0
                    continue
                
                # Sort unique values
                sorted_vals = sorted(values)
                
                # Find minimum difference
                min_diff = float('inf')
                for t in range(1, len(sorted_vals)):
                    min_diff = min(min_diff, sorted_vals[t] - sorted_vals[t - 1])
                
                ans[i][j] = min_diff
        
        return ans
