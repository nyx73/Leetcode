## 📌 Problem Summary
You are given:
- `positions[]`: positions of robots on a line
- `healths[]`: health of each robot
- `directions`: movement direction ('L' or 'R')

### ⚡ Rules:
- All robots move at the same speed
- Collision happens when two robots meet
- If collision occurs:
  - Lower health robot is removed
  - Higher health robot loses 1 health and continues
  - If equal health → both removed

---

## 🎯 Goal
Return the health of **surviving robots** in the **original input order**.

---

## 🧠 Key Idea (Stack Approach)

### 💡 Why Stack?
- Only **R → L collisions** matter
- Track robots moving right using a stack
- When a left-moving robot comes, resolve collisions

---

## 🚀 Approach

1. Combine data into tuples:
   ```
   (position, health, direction, original_index)
   ```

2. Sort robots by position

3. Use a stack to store **right-moving robots**

4. For each robot:
   - If moving right → push to stack
   - If moving left → resolve collisions with stack

---

## ⚙️ Collision Logic

While stack not empty and top is 'R':
- Compare healths:
  - If stack robot < current → remove stack robot, current -= 1
  - If equal → both removed
  - If stack robot > current → stack robot -= 1, current removed

---

## 🧾 Python Code

```python
class Solution:
    def survivedRobotsHealths(self, positions, healths, directions):
        robots = sorted(
            [(positions[i], healths[i], directions[i], i) for i in range(len(positions))]
        )

        stack = []

        for pos, health, direction, idx in robots:
            if direction == 'R':
                stack.append([pos, health, direction, idx])
            else:
                while stack and stack[-1][2] == 'R' and health > 0:
                    if stack[-1][1] < health:
                        health -= 1
                        stack.pop()
                    elif stack[-1][1] == health:
                        stack.pop()
                        health = 0
                        break
                    else:
                        stack[-1][1] -= 1
                        health = 0
                        break
            
                if health > 0:
                    stack.append([pos, health, direction, idx])

        # Sort by original index
        stack.sort(key=lambda x: x[3])

        return [robot[1] for robot in stack]
```

---

## 📊 Complexity

| Type | Value |
|------|------|
| Time | O(n log n) (sorting) |
| Space | O(n) |

## 🧪 Example

### Input:
```
positions = [3,5,2,6]
healths = [10,10,15,12]
directions = "RLRL"
```

### Output:
```
[14]
```
