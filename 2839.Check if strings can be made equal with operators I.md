# Swap Distance-2 Problem (Strings of Length 4)

## 🧩 Problem Statement

You are given two strings `s1` and `s2`, each of length **4**,
consisting of lowercase English letters.

### 🔁 Allowed Operation

You may perform the following operation any number of times on **either
string**:

-   Choose indices `i` and `j` such that:

        j - i = 2

-   Swap characters at those positions.

------------------------------------------------------------------------

## 🎯 Goal

Return **true** if you can make `s1` equal to `s2`, otherwise return
**false**.

------------------------------------------------------------------------

## 🧪 Examples

### Example 1

    Input:
    s1 = "abcd"
    s2 = "cdab"

    Output:
    true

### Explanation

    Step 1: swap (0,2) → "cbad"
    Step 2: swap (1,3) → "cdab"

------------------------------------------------------------------------

### Example 2

    Input:
    s1 = "abcd"
    s2 = "dacb"

    Output:
    false

------------------------------------------------------------------------

## 💡 Key Insight

The operation only allows swaps between:

-   index `0 ↔ 2`
-   index `1 ↔ 3`

So indices form **two independent groups**:

  Group            Indices
  ---------------- ---------
  Even positions   0, 2
  Odd positions    1, 3

👉 You can rearrange characters **only within each group**, not across
groups.

------------------------------------------------------------------------

## 🧠 Approach

1.  Extract characters from:

    -   Even indices → `[0, 2]`
    -   Odd indices → `[1, 3]`

2.  Sort both groups for `s1` and `s2`

3.  Compare:

    -   Even groups must match
    -   Odd groups must match

------------------------------------------------------------------------

## 🧪 Python Solution

``` python
class Solution:
    def canBeEqual(self, s1: str, s2: str) -> bool:
        s1_even = sorted([s1[0], s1[2]])
        s1_odd = sorted([s1[1], s1[3]])

        s2_even = sorted([s2[0], s2[2]])
        s2_odd = sorted([s2[1], s2[3]])

        return s1_even == s2_even and s1_odd == s2_odd
```

------------------------------------------------------------------------

## ⏱ Complexity

  Operation                 Complexity
  ------------------------- ------------
  Sorting (constant size)   O(1)
  Total                     **O(1)**

------------------------------------------------------------------------

## 🚀 Summary

-   Only swaps at distance 2 are allowed
-   Treat indices as **two independent buckets**
-   Compare sorted buckets

------------------------------------------------------------------------

## 🎉 Final Thought

This is a great example of recognizing **constraints → grouping →
simplification**.
