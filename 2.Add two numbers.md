# 🔢 Add Two Numbers (Linked List) - Full Explanation

## ✅ Problem Statement

You are given two non-empty linked lists representing two non-negative integers.  
The digits are stored in **reverse order**, and each node contains a single digit.  
Return the sum of the two numbers as a linked list.

### ✨ Example 1:
- Input: `l1 = [2,4,3]`, `l2 = [5,6,4]`
- Output: `[7,0,8]`
- Explanation: `342 + 465 = 807`

### ✨ Example 2:
- Input: `l1 = [0]`, `l2 = [0]`
- Output: `[0]`

### ✨ Example 3:
- Input: `l1 = [9,9,9,9,9,9,9]`, `l2 = [9,9,9,9]`
- Output: `[8,9,9,9,0,0,0,1]`

---

## 📌 Constraints

- `1 <= list length <= 100`
- `0 <= Node.val <= 9`
- Input lists do not have leading zeroes, **except 0 itself**

---

## 🧑‍💻 Python Code

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def addTwoNumbers(self, list1, list2):
        dummy_head = ListNode()
        current = dummy_head
        carry = 0

        while list1 or list2 or carry:
            sum_ = (list1.val if list1 else 0) + (list2.val if list2 else 0) + carry
            carry, val = divmod(sum_, 10)

            current.next = ListNode(val)
            current = current.nextcarry = 0

            list1 = list1.next if list1 else None
            list2 = list2.next if list2 else None

        return dummy_head.next
```
---

## 🧱 1. Create a dummy node and pointer:
```python
dummy_head = ListNode()
current = dummy_head
```
- This dummy node helps us build the result linked list.
- current is used to track where to insert the next digit node.

## 🔢 2. Initialize carry:
```python
carry = 0
```
-carry holds the overflow when the sum of digits ≥ 10.

## 🔁 3. Traverse both linked lists:
  ```python
while list1 or list2 or carry:
```
-Continue the loop while there's something to process:

  - Remaining digits in list1
  - Remaining digits in list2
  - Non-zero carry

## ➕ 4. Add corresponding digits and carry:
```python
sum_ = (list1.val if list1 else 0) + (list2.val if list2 else 0) + carry
```
- Get the digit from each list node (or 0 if it’s None).
- Add them together with the carry from the previous step.
