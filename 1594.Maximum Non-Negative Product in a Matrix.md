# Maximum Non-Negative Product in a Matrix Path

## 🧩 Problem

You are given an `m x n` matrix `grid`.  
You start at the top-left corner `(0, 0)` and can only move:

- Right ➡️
- Down ⬇️

Your goal is to reach `(m - 1, n - 1)` with a path that gives the **maximum non-negative product**.

### 🔁 Rules
- The product is the multiplication of all values along the path.
- Return the result modulo `10^9 + 7`.
- If the maximum product is negative, return `-1`.

---

## 💡 Key Insight

Because the grid contains **negative numbers**, the product can flip signs:

- Negative × Negative = Positive ✅  
- Positive × Negative = Negative ❌  

👉 So at each cell, we must track:
- **Maximum product**
- **Minimum product**

Why?  
A small negative value can become a large positive value later!

---

## 🧠 Dynamic Programming Approach

We maintain two DP tables:

- `max_dp[i][j]` → Maximum product up to `(i, j)`
- `min_dp[i][j]` → Minimum product up to `(i, j)`
