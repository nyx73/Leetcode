# Longest Common Prefix (LCP) Matrix → Construct String

## 🧩 Problem Statement

We define the **LCP (Longest Common Prefix) matrix** of a string `word`
of length `n` as an `n x n` matrix where:

-   `lcp[i][j]` = length of the longest common prefix between:
    -   substring `word[i...n-1]`
    -   substring `word[j...n-1]`

### 🎯 Task

Given an `n x n` matrix `lcp`, return the **lexicographically smallest
string** `word` that produces this matrix.

If no such string exists → return `""`.

------------------------------------------------------------------------

## 📌 Lexicographical Order

A string `a` is smaller than `b` if at the first differing position:

-   `a[i] < b[i]`

Example:

    "aabd" < "aaca"

------------------------------------------------------------------------

## 🧪 Examples

### Example 1

    Input:
    lcp = [
     [4,0,2,0],
     [0,3,0,1],
     [2,0,2,0],
     [0,1,0,1]
    ]

    Output:
    "abab"

------------------------------------------------------------------------

### Example 2

    Input:
    lcp = [
     [4,3,2,1],
     [3,3,2,1],
     [2,2,2,1],
     [1,1,1,1]
    ]

    Output:
    "aaaa"

------------------------------------------------------------------------

### Example 3

    Input:
    lcp = [
     [4,3,2,1],
     [3,3,2,1],
     [2,2,2,1],
     [1,1,1,3]
    ]

    Output:
    ""

------------------------------------------------------------------------

## ⚠️ Observations

1.  `lcp[i][i] = n - i` must always hold.
2.  If `lcp[i][j] > 0`, then:
    -   `word[i] == word[j]`
3.  If `lcp[i][j] == k`, then:
    -   `word[i + k] != word[j + k]` (if within bounds)

------------------------------------------------------------------------

## 💡 Approach

### Step 1: Validate diagonal

Ensure:

    lcp[i][i] == n - i

------------------------------------------------------------------------

### Step 2: Build groups (Union-Find / Greedy)

If `lcp[i][j] > 0`, then `i` and `j` must have the same character.

------------------------------------------------------------------------

### Step 3: Assign characters

-   Assign smallest characters (`'a'`, `'b'`, ...)
-   Ensure at most 26 groups

------------------------------------------------------------------------

### Step 4: Validate LCP

Recompute LCP from constructed string and compare.

------------------------------------------------------------------------

## 🧠 Python Solution

``` python
class Solution:
    def findTheString(self, lcp):
        n = len(lcp)

        # Step 1: Check diagonal
        for i in range(n):
            if lcp[i][i] != n - i:
                return ""

        parent = list(range(n))

        def find(x):
            if parent[x] != x:
                parent[x] = find(parent[x])
            return parent[x]

        def union(x, y):
            px, py = find(x), find(y)
            if px != py:
                parent[py] = px

        # Step 2: Union positions
        for i in range(n):
            for j in range(n):
                if lcp[i][j] > 0:
                    union(i, j)

        # Step 3: Assign characters
        groups = {}
        char = 'a'
        res = [''] * n

        for i in range(n):
            root = find(i)
            if root not in groups:
                if char > 'z':
                    return ""
                groups[root] = char
                char = chr(ord(char) + 1)
            res[i] = groups[root]

        word = "".join(res)

        # Step 4: Validate LCP
        dp = [[0]*n for _ in range(n)]

        for i in reversed(range(n)):
            for j in reversed(range(n)):
                if word[i] == word[j]:
                    if i+1 < n and j+1 < n:
                        dp[i][j] = dp[i+1][j+1] + 1
                    else:
                        dp[i][j] = 1

        if dp != lcp:
            return ""

        return word
```

------------------------------------------------------------------------

## ⏱ Complexity

  Step         Complexity
  ------------ ------------
  Union-Find   O(n² α(n))
  Validation   O(n²)
  Total        **O(n²)**

------------------------------------------------------------------------

## 🚀 Key Insight

The matrix **forces equality relationships** between indices.

👉 Treat indices like nodes in a graph\
👉 Merge them into groups\
👉 Assign smallest possible characters

------------------------------------------------------------------------

## 🎉 Summary

-   Use Union-Find to group indices
-   Assign lexicographically smallest characters
-   Validate using DP
