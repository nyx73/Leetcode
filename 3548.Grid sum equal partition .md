# 🧩 Equal Sumgrid partition II

## 📘 Problem Statement

You are given an `m x n` matrix `grid` of positive integers. Your task
is to determine if it is possible to make either one horizontal or one
vertical cut on the grid such that:

-   Each of the two resulting sections formed by the cut is non-empty.
-   The sum of elements in both sections is equal, OR can be made equal
    by removing at most one cell.
-   If a cell is removed, the remaining section must stay **connected**.

A section is connected if every cell can be reached from any other cell
using up/down/left/right moves.

------------------------------------------------------------------------

## 🧠 Approach

1.  Try all possible:
    -   Horizontal cuts
    -   Vertical cuts
2.  For each cut:
    -   Compute sums of both sections
    -   If equal → return True
    -   Else:
        -   Find difference `diff = abs(sum1 - sum2)`
        -   Check if we can remove a cell with value `diff`
3.  Connectivity rule:
    -   If section is 2D → always safe
    -   If section is 1D → only edge removal allowed

------------------------------------------------------------------------

## 🚀 Python Solution

``` python
from collections import Counter

class Solution:
    def canPartitionGrid(self, grid):
        m, n = len(grid), len(grid[0])

        total = sum(sum(row) for row in grid)

        row_prefix = [0] * m
        row_prefix[0] = sum(grid[0])
        for i in range(1, m):
            row_prefix[i] = row_prefix[i - 1] + sum(grid[i])

        col_prefix = [0] * n
        for j in range(n):
            col_prefix[j] = sum(grid[i][j] for i in range(m))
        for j in range(1, n):
            col_prefix[j] += col_prefix[j - 1]

        top_counter = Counter()
        bottom_counter = Counter()

        for row in grid:
            bottom_counter.update(row)

        for i in range(m - 1):
            top_counter.update(grid[i])
            for val in grid[i]:
                bottom_counter[val] -= 1
                if bottom_counter[val] == 0:
                    del bottom_counter[val]

            sum1 = row_prefix[i]
            sum2 = total - sum1

            if sum1 == sum2:
                return True

            diff = abs(sum1 - sum2)

            if sum1 > sum2:
                if diff in top_counter:
                    if i + 1 > 1 and n > 1:
                        return True
                    if i + 1 == 1:
                        if grid[0][0] == diff or grid[0][-1] == diff:
                            return True
                    if n == 1:
                        if grid[0][0] == diff or grid[i][0] == diff:
                            return True
            else:
                if diff in bottom_counter:
                    if m - i - 1 > 1 and n > 1:
                        return True
                    if m - i - 1 == 1:
                        if grid[i+1][0] == diff or grid[i+1][-1] == diff:
                            return True
                    if n == 1:
                        if grid[i+1][0] == diff or grid[-1][0] == diff:
                            return True

        left_counter = Counter()
        right_counter = Counter()

        for r in range(m):
            for c in range(n):
                right_counter[grid[r][c]] += 1

        for j in range(n - 1):
            for r in range(m):
                val = grid[r][j]
                left_counter[val] += 1
                right_counter[val] -= 1
                if right_counter[val] == 0:
                    del right_counter[val]

            sum1 = col_prefix[j]
            sum2 = total - sum1

            if sum1 == sum2:
                return True

            diff = abs(sum1 - sum2)

            if sum1 > sum2:
                if diff in left_counter:
                    if j + 1 > 1 and m > 1:
                        return True
                    if j + 1 == 1:
                        if grid[0][0] == diff or grid[-1][0] == diff:
                            return True
                    if m == 1:
                        if grid[0][0] == diff or grid[0][j] == diff:
                            return True
            else:
                if diff in right_counter:
                    if n - j - 1 > 1 and m > 1:
                        return True
                    if n - j - 1 == 1:
                        if grid[0][j+1] == diff or grid[-1][j+1] == diff:
                            return True
                    if m == 1:
                        if grid[0][j+1] == diff or grid[0][-1] == diff:
                            return True

        return False
```

------------------------------------------------------------------------

## ✅ Complexity

-   Time: **O(m × n)**
-   Space: **O(m × n)**

------------------------------------------------------------------------

## 🎯 Summary

-   Try all cuts
-   Compare sums
-   Remove one cell if needed
-   Ensure connectivity
-   Return True if any valid partition exists
